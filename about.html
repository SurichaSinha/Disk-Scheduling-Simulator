About Our Operating System Project

Disk Scheduling Algorithms Simulation

This project offers a simulation of various disk scheduling algorithms, designed to deepen understanding of disk I/O management. Our system enables users to explore techniques such as FCFS, SSTF, SCAN, C-SCAN, LOOK, and C-LOOK through an interactive and visually engaging interface.

Project Features

  • Simulated Engine (PHP): Manages the core logic for scheduling and processing disk requests efficiently.
  • User Interface (HTML & CSS): Delivers a clean, responsive design to enhance user interaction.
  • Visualization (JavaScript): Illustrates disk head movements dynamically for improved comprehension.
