About Our Operating System Project

Disk Scheduling Algorithms Simulation

This project is a simulation of various disk scheduling algorithms, aiming to enhance understanding of how they work in managing disk I/O operations. Our system allows users to visualize different scheduling techniques such as FCFS, SSTF, SCAN, C-SCAN, LOOK and C-LOOK through an interactive interface.

Project Features

  • Simulated Engine (PHP): Handles the core logic for scheduling and processing disk requests.
  • User Interface (HTML & CSS): Provides a clean and responsive design for user interaction.
  • Visualization (JavaScript): Displays the movement of the disk head dynamically for better comprehension.
